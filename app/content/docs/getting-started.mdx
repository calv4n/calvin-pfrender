## Folder structure

Store documentation files in `app/content/docs`.

## Recommended pattern

Use one file per topic and add frontmatter with at least a `title`.

```mdx
---
title: My Topic
description: Short summary
---

## Section

Write your content here.
```

## Route mapping

`content/docs/getting-started.mdx` becomes `/docs/getting-started`.
