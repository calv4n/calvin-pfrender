## Overview

This docs site is mounted at `/docs` and reads content from `content/docs`.

## Add more pages

Create more `.md` or `.mdx` files in `content/docs` and they will appear in the sidebar.

## Search

Built-in search is available through the Fumadocs search dialog and is backed by `/api/search`.
